BLOCK ELEMENTS

div
defines a division or a section in an HTML document align, title
<div>...</div>
h1
The H1 to H6 tags are used to define HTML headings align
<h1>Title</h1>
footer
Specifies the trailing area (footer) of a document or section
<...> <footer>content</footer>
main
specifies the main content of a document
<...> <main></main>
